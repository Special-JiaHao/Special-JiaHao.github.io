# 线性回归

# 线性模型

回归类算法。对给定的训练集进行线性拟合，从而求得一条能使绝大多数样本点尽可能被准确的预测的拟合线，即：

f(x)=w1x1+w2x2+⋯+wnxn+bf(x) = w_{1}x_{1} + w_{2}x_{2} + \cdots + w_{n}x_{n} + bf(x)=w1​x1​+w2​x2​+⋯+wn​xn​+b

其中样本特征的维度为nnn, wi(i∈1,2,...,n)、bw_{i}(i\in{1, 2, ..., n})、bwi​(i∈1,2,...,n)、b 为拟合直线的 权重 和 偏置 ，也是我们需要求解的变量。权重决定了这个特征对我们预测值的影响程度。线性模型的本质是一个仿射变换，通过加权和对特征进行线性变换，并通过偏置来进行平移.
